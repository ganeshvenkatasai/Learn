Class Diagram :

A class diagram consists of:

Classes → ClassName
Interfaces → <<interface>> InterfaceName
Abstract Classes → <<abstract>> ClassName 
Relationships → Associations, Aggregation, Composition, Inheritance, Dependency.
Access Modifiers:
+ (Public) → Accessible anywhere.
- (Private) → Accessible only within the class.
# (Protected) → Accessible in the class and its subclasses.
~ (Package-private) → Accessible in the same package.

------------------------
| ClassName            |
------------------------
| - attribute1: Type   |
------------------------
| + method1(): Return  |
------------------------
